# 淨零轉型實務工具箱（Net-Zero Transition Playbook）

## 定位

「淨零（Net-Zero）」是 2050 年前後多數國家的法定或承諾目標。對台灣企業而言，淨零轉型的實務壓力來自：
- **國家政策**：《氣候變遷因應法》、碳費、能源轉型
- **國際客戶**：Apple、Nike、Walmart 等要求供應鏈減碳
- **資本市場**：ISSB 揭露、ESG 投資流向
- **貿易壁壘**：歐盟 CBAM、美國清潔競爭法案
- **融資條件**：綠色貸款、永續連結貸款

EMBA 學員面對的核心問題通常是：「**我該怎麼做？從哪開始？花多少錢？多久見效？**」本 skill 提供從「盤查 → 目標 → 路徑 → 揭露 → 執行 → 驗證」的完整流程。

**與相近 Asgard skill 的邊界**：
- `grad-sustainability` — 永續理論層（TBL、SDG、ESG、循環經濟）
- **本 skill** — **實務執行**（盤查方法、SBTi 申請、TCFD 揭露、供應鏈減碳、台灣碳費）

## 何時使用

**觸發條件**
- 首次啟動碳盤查
- 設計 2030 / 2050 減碳目標
- TCFD / ISSB / 永續報告書撰寫
- 回應國際客戶減碳問卷（CDP、EcoVadis）
- 供應鏈碳管理系統建置
- 碳費應對與財務影響分析
- CBAM、清潔競爭法案等貿易壁壘應對
- 綠電採購、再生能源憑證
- EMBA ESG、營運、策略課程個案分析

**不適用**
- 純理論層的永續策略 → Asgard `grad-sustainability`
- 財務報表分析 → Asgard `biz-financial-ratios`、`data-financial-analysis`
- ESG 投資組合設計 → 另找投資分析工具
- 個別技術（如熱泵、儲能）深度 → 需產業工程團隊

## IRON LAW — 淨零轉型的三條鐵律

```
IRON LAW 1：沒有盤查就沒有減碳
多數企業先喊「2050 淨零」才補盤查，結果目標與實況脫節。
正確順序：
範疇 1/2 盤查（6–12 個月） → 範疇 3 熱點識別 →
基準年確定 → 目標設定（SBTi 對齊） →
路徑規劃 → 揭露 → 執行 → 年度驗證
```

```
IRON LAW 2：範疇 3 才是真正的戰場
多數製造業範疇 3（上下游）占總碳排 70–95%。
只減範疇 1（直接）+ 範疇 2（電力）無法達成淨零。
供應鏈參與（Scope 3 Engagement）是關鍵且最困難的環節。
```

```
IRON LAW 3：減碳是財務決策，不只是永續決策
沒有內部碳定價（Internal Carbon Pricing）的組織，
永續部門提什麼都是「花錢」；
有內部碳定價的組織，減碳變成「CFO 的投資組合問題」。
淨零轉型的治理高度 = CFO 深度介入 + 董事會納入議程。
```

## 流程總覽

```
Step 1：溫室氣體盤查（GHG Inventory）
  ├─ 範疇 1：直接排放（鍋爐、車輛、製程、冷媒）
  ├─ 範疇 2：外購電力（市場基礎 / 位置基礎）
  └─ 範疇 3：價值鏈（15 大類、通常占 70%+）

Step 2：熱點分析與目標設定
  ├─ 熱點識別（Material Categories）
  ├─ 基準年 / 目標年確定
  └─ SBTi 對齊（1.5°C / Well-below 2°C）

Step 3：減碳路徑（Decarbonization Pathway）
  ├─ 能源效率
  ├─ 電氣化
  ├─ 再生能源（自發自用、PPA、T-REC）
  ├─ 製程革新
  ├─ 供應鏈參與
  └─ 碳移除（Offset / Removal，最後手段）

Step 4：揭露（TCFD → ISSB S1/S2）
  ├─ 治理、策略、風險管理、指標與目標
  └─ 情境分析（1.5°C / 2°C / 4°C）

Step 5：內部治理
  ├─ 內部碳定價
  ├─ 永續 KPI 連結薪酬
  ├─ 董事會與永續委員會
  └─ 跨部門協作機制

Step 6：驗證與溝通
  ├─ 第三方查證（ISO 14064-3）
  ├─ 客戶問卷回應（CDP、EcoVadis）
  └─ 投資人溝通（永續報告書、法說會）
```

## Step 1：溫室氣體盤查

### 主要標準對照

| 標準 | 範圍 | 適用 | 台灣現況 |
|---|---|---|---|
| GHG Protocol | 企業 / 產品 | 國際最廣 | 台灣多數採用 |
| ISO 14064-1 | 組織層 | 可驗證、配合 ISO 14064-3 | 環保署指定 |
| ISO 14067 | 產品碳足跡 | 產品層 | 國際客戶要求 |
| PAS 2050 | 產品 | 英國標準 | 較少採用 |

### 三大範疇

**範疇 1（Scope 1）：直接排放**
- 廠區燃燒：鍋爐、柴油機、熔爐
- 移動源：公司車輛
- 製程：化學反應直接排放
- 逸散：冷媒、SF6、滅火劑

**範疇 2（Scope 2）：外購能源間接排放**
- 外購電力
- 外購蒸汽、熱、冷
- **兩種計算方法**：
  - **位置基礎（Location-based）**：用電網平均排放係數
  - **市場基礎（Market-based）**：考慮綠電、T-REC

**範疇 3（Scope 3）：價值鏈其他間接排放（15 大類）**

| 類別 | 典型涵蓋 |
|---|---|
| 1. 採購商品與服務 | 原物料、零組件、服務 |
| 2. 資本財 | 設備、廠房 |
| 3. 燃料與能源相關活動 | 上游採礦、輸配 |
| 4. 上游運輸 | 進料運輸 |
| 5. 廢棄物處理 | 垃圾、廢水 |
| 6. 商務差旅 | 飛機、高鐵、汽車 |
| 7. 員工通勤 | 通勤交通工具 |
| 8. 上游租賃資產 | 租賃設施 |
| 9. 下游運輸 | 出貨運輸 |
| 10. 銷售產品加工 | 客戶再加工排放 |
| 11. 銷售產品使用 | 產品使用期間排放 |
| 12. 銷售產品報廢 | 產品壽命結束處理 |
| 13. 下游租賃資產 | 出租設施 |
| 14. 加盟 | 加盟店排放 |
| 15. 投資 | 投資組合碳排 |

### 盤查流程

1. **組織邊界**：營運控制 / 財務控制 / 股權比例
2. **活動數據蒐集**：能源、差旅、廢棄物、採購
3. **排放係數選用**：GHG Protocol / ISO / IPCC / 在地資料庫
4. **計算與彙總**：活動數據 × 排放係數
5. **不確定性分析**：主要排放源的誤差範圍
6. **第三方查證**（選擇性）：ISO 14064-3

### 常見盤查陷阱
- **範疇邊界模糊**：子公司、合資、委外製造是否納入？
- **範疇 2 計算方法未明**：位置 vs. 市場基礎需一致
- **範疇 3 跳過困難類別**：最大（類別 1、11）往往最難計算，易被忽略
- **係數版本錯位**：IPCC、台灣電網係數每年更新

## Step 2：目標設定（SBTi 對齊）

### 為何要 SBTi

Science Based Targets initiative 是國際最具公信力的企業減碳目標驗證機制，要求：
- **科學對齊**（1.5°C 或 Well-below 2°C）
- **涵蓋範疇 1+2+3**（範疇 3 若占 > 40% 必須設定）
- **時間尺度**（通常 5–15 年）
- **定期進度報告**

### SBTi 申請四步驟

1. **Commitment**（承諾）：提交承諾書、6 個月內開始設目標
2. **Develop Target**（制訂）：依 SBTi 方法計算
3. **Submit for Validation**（提交驗證）：SBTi 審核 30 天
4. **Communicate & Disclose**（揭露溝通）：公開目標、年度報告進度

### 目標類型

**絕對減量（Absolute Reduction）**
- 如：2030 年範疇 1+2 較 2020 減少 42%
- 適合：排放穩定 / 下降的成熟企業

**強度目標（Intensity Reduction）**
- 如：每單位營收碳排 2030 減 50%
- 適合：高成長企業
- **注意**：強度下降 ≠ 絕對下降

**淨零目標（Net-Zero）**
- 2050 年範疇 1+2+3 達到淨零
- 需配合「近期目標」（2030）與「長期目標」（2050）

### SBTi 各產業路徑
- 一般企業：Cross-sector
- 高排產業（水泥、鋼鐵、化工、電力）有專屬路徑
- 金融業：PCAF + SBTi for Financial Institutions

## Step 3：減碳路徑設計

### 典型企業減碳槓桿（依優先順序）

#### 1. 能源效率
- 製程最佳化
- LED、變頻空調
- 壓縮空氣、熱回收
- 能源管理系統（ISO 50001）
- **投資報酬率**：通常 2–5 年回本

#### 2. 電氣化
- 熱泵取代鍋爐
- 電動車 / 堆高機
- 電力加熱取代燃油
- **前提**：電力來源綠化

#### 3. 再生能源
- **自發自用**：屋頂太陽能
- **PPA（Power Purchase Agreement）**：直接與綠電業者簽約
- **T-REC（台灣再生能源憑證）**：購買憑證抵充
- **綠電採購策略**：RE100 要求 100% 綠電

#### 4. 製程革新
- 低碳原料替代
- 高效設備更新
- 氫能 / 低碳燃料
- CCUS（碳捕集、利用與封存）

#### 5. 供應鏈參與（Scope 3）
- 供應商碳數據蒐集
- 設定供應商減碳目標
- 低碳供應商優先採購
- 共同減碳計畫

#### 6. 產品設計
- 產品使用期減碳（提升能效）
- 循環經濟設計（耐用、可修、可回收）
- 材料替代

#### 7. 碳移除與抵換（最後手段）
- 植樹、造林
- 自然碳匯
- 工程碳移除（DAC 等新技術）
- **原則**：抵換不應取代實質減碳

### 減碳路徑視覺化（MACC 曲線）

**邊際減碳成本曲線（Marginal Abatement Cost Curve）**：
- 橫軸：減碳量（tCO2e）
- 縱軸：每噸減碳成本（元 / tCO2e）
- 從最低成本的槓桿開始

**台灣企業常見 MACC 起點**：
- 最划算：LED 照明、變頻空調（負成本）
- 中等：熱回收、屋頂太陽能
- 昂貴：綠電 PPA、製程革新
- 最貴：碳移除、低碳原料

## Step 4：揭露（TCFD → ISSB S1/S2）

### TCFD 四大支柱
1. **治理**：董事會 / 管理階層的氣候職責
2. **策略**：氣候風險與機會對業務的影響
3. **風險管理**：氣候風險識別、評估、管理流程
4. **指標與目標**：碳排、目標、財務指標

### ISSB S1 / S2（2024 起）
- **S1**：一般永續相關財務資訊揭露
- **S2**：氣候相關揭露（取代 TCFD）
- **台灣採用時程**：金管會規劃 2026 大型、2029 全面

### 情境分析（Scenario Analysis）

**IPCC 情境**：
- 1.5°C（Paris-aligned）
- 2°C（Well-below 2）
- 3–4°C（Business as usual）

**分析面向**：
- 實體風險（Physical）：極端氣候、海平面、颱風
- 轉型風險（Transition）：政策、技術、市場、法律
- 機會（Opportunity）：新市場、效率、資源

### 台灣永續報告書

**主要規範**：
- 實收資本額 20 億以上上市公司強制
- GRI 準則 + SASB 產業指標 + TCFD
- 2027 起 ISSB S1/S2 導入

**報告書核心章節**：
1. 公司概況
2. 永續管理（治理）
3. 環境（碳、水、廢棄物、生物多樣性）
4. 社會（員工、供應鏈、社區）
5. 治理（董事會、風險、倫理）
6. 附錄：GRI / SASB / TCFD 索引、外部保證

## Step 5：內部治理

### 內部碳定價（Internal Carbon Pricing）

**三種模式**：
- **Shadow Price**：用於投資決策評估（不實際收費）
- **Carbon Fee**：內部各部門實際收費、資金專款專用
- **Implicit Price**：減碳投資組合的邊際成本

**台灣企業典型起點**：NT$ 300–1,500 / tCO2e（視產業）
**國際大企業**：USD 30–100 / tCO2e

### 永續薪酬連結

**設計要點**：
- 高管年度變動薪酬中 10–20% 與永續 KPI 連結
- KPI 可量化、與集團目標對齊
- 範圍 1+2 優先、範疇 3 進階

### 永續委員會

- 董事會層級
- 永續長（CSO）為幕僚
- 跨部門整合（財務、營運、採購、人資）
- 季度審議

### 組織設計
- **永續辦公室**：獨立部門（3–10 人，視規模）
- **各部門永續代表**：由既有主管兼任
- **投資人關係結合**：IR + ESG 整合

## Step 6：驗證與溝通

### 第三方查證
- ISO 14064-3 盤查查證
- 永續報告書保證（有限 / 合理保證）
- SBTi 官方驗證
- 客戶審核（CDP、EcoVadis、Apple Supplier Responsibility）

### 客戶問卷常見項目
- 碳盤查結果
- 減碳目標（SBTi）
- 綠電比例
- 永續政策
- 供應鏈管理
- 違規紀錄

### 投資人溝通
- 法說會永續專題
- ESG 評比（MSCI、S&P、Sustainalytics）
- 永續連結貸款 / 綠色債券

## 台灣特殊議題

### 碳費制度（2026 起徵）
- 主管機關：環境部
- 適用對象：年排 > 2.5 萬噸 CO2e 的排放源
- 費率：2026 起每噸 300 元起（依自主減量 / 高碳洩漏風險調整）
- 徵收基礎：範疇 1 + 範疇 2

### 自主減量計畫
- 企業提出自主減量計畫獲核准可適用優惠費率
- 需達成 2030 年特定減量幅度（依產業別）

### 再生能源憑證（T-REC）
- 台灣綠電交易平台
- 一張 = 1,000 度
- 供需緊繃（半導體業主導需求）
- 2025–2026 價格：NT$ 4–6 / 度

### CBAM（歐盟碳邊境調整機制）
- 2026 開始徵收
- 適用：鋼鐵、鋁、水泥、肥料、電力、氫
- 進口商須申報產品碳含量、購買 CBAM 憑證
- 台灣相關產業：鋼鐵、水泥、鋁加工出口

### 清潔競爭法案（美國）
- 2027 上路
- 適用：鋼鐵、鋁、水泥、肥料、玻璃、紙、乙烯等
- 台灣相關產業需關注

### RE100 壓力
- Apple、Google、Microsoft 等國際大客戶 100% 綠電承諾
- 對台灣供應鏈（半導體、電子代工、面板）直接壓力
- 台積電、台達電等已自行加入 RE100

## 分析流程

```
Step 1：公司現況診斷
  - 現有盤查結果（若有）
  - 產業特性與碳排熱點
  - 客戶 / 監管壓力

Step 2：差距分析
  - 與 SBTi 對齊需求
  - 與同業標竿
  - 與國際客戶要求

Step 3：減碳路徑方案
  - 3–5 條可能路徑
  - MACC 分析
  - 投資需求

Step 4：治理結構建議
  - 永續委員會 / 辦公室
  - 內部碳定價
  - 薪酬連結

Step 5：揭露計畫
  - TCFD / ISSB 導入
  - 永續報告書
  - 客戶問卷

Step 6：執行時程與里程碑
  - 短期（0–12 月）
  - 中期（1–3 年）
  - 長期（3–10 年）
```

## Examples

### 正確應用
**情境**：中部金屬加工業上市公司（營收 45 億、範疇 1+2 約 85,000 tCO2e、出口歐美占 60%）被 Apple 要求 2030 年前 100% 綠電，同時面對 CBAM 與台灣碳費。

**診斷重點**：
- 盤查現況：範疇 1+2 已完成、範疇 3 僅類別 1 初估
- 碳排熱點：範疇 2（外購電力）占 72%、範疇 1（製程 + 天然氣）占 28%
- 客戶壓力：Apple 要求 2030 綠電、EcoVadis 評等 Silver
- 監管壓力：2026 起碳費年支出預估 2,550 萬；CBAM 影響出口歐盟部分

**減碳路徑建議**：
1. 立即：屋頂太陽能（3 MW、減 3% 範疇 2）
2. 中期：綠電 PPA 20% + 熱回收 + 製程效率（2028 前減 15%）
3. 長期：2030 年 100% 綠電（PPA 70% + T-REC 30%）
4. 範疇 3：前 20 家供應商逐步要求盤查與減碳目標

**治理建議**：
- 永續委員會董事會層級
- 內部碳定價起始 NT$ 800 / tCO2e
- 高管年度變動薪酬 10% 與減碳 KPI 連結
- TCFD 2026 年報起揭露、ISSB S2 2028 導入

**投資需求**：
- 3 年總投資 NT$ 2.8 億（含 PPA 保證金、太陽能設施、能源管理系統、數位平台）
- 碳費支出 2026–2030 累計 NT$ 1.5 億
- 可爭取永續連結貸款 NT$ 5 億、利率優惠 0.15%

### 錯誤應用
- 宣稱「2030 淨零」但未做範疇 3 盤查 → 空口承諾、漂綠風險
- 購買大量碳抵換取代實質減碳 → SBTi 拒絕、客戶質疑
- 永續報告書寫漂亮但董事會無實質議程 → 揭露不實
- 忽略 CBAM → 出口歐盟產品競爭力受損

## Gotchas

- **範疇 3 的「80/20 陷阱」**：多數企業範疇 3 類別 1（採購）+ 類別 11（產品使用）占 80%。其他 13 類可簡略處理、但不可完全跳過（SBTi 要求涵蓋 67% 以上）
- **綠電的「加性原則」（Additionality）**：購買舊電廠綠電憑證 ≠ 實質減碳。SBTi 與國際標準日益嚴格，PPA 需要搭配新建再生能源才被認可
- **淨零與碳中和的混淆**：「淨零」需大幅實質減碳 + 少量不可避免排放的碳移除；「碳中和」可能靠大量抵換達成。市場與客戶對後者日益不買單
- **TCFD 情境分析的「紙上談兵」**：多數公司情境分析未實際影響策略決策。要實質化需：高管參與、量化影響、納入資本配置
- **內部碳定價的「形式化」**：定了價但沒收費、沒影響投資決策 = 沒效果。關鍵是讓 CFO 介入、讓投資案的 IRR 計算含碳成本
- **SBTi 提交前的「逆向驗算」**：先用 SBTi 工具估目標、再倒推可行性。直接承諾後再算常發現目標不可達
- **碳費的財務影響常被低估**：不只是當期支出，還包括對客戶價格、庫存成本、長期競爭力。需納入 3–5 年財務預測
- **綠電採購的「排他性」風險**：T-REC 簽約要求排他性，若綠電產能供不應求，早簽者有優勢。拖延反而吃虧
- **供應鏈減碳的「誰付錢」爭議**：要求供應商減碳但不給溢價 = 供應商陽奉陰違。需設計「低碳採購溢價」機制
